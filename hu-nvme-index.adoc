= 了解如何使用 NVMe-oF 配置 ONTAP SAN 主机
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以将支持的 SAN 主机配置为使用 NVMe over Fabrics (NVMe-oF) 协议，并将 ONTAP 作为存储目标。NVMe-oF 包括 NVMe over Fibre Channel (NVMe/FC) 和 NVMe over TCP (NVMe/TCP)。根据主机操作系统 (OS) 和 ONTAP 版本，您可以在主机操作系统上配置和验证 NVMe/FC 或 NVMe/TCP 协议，或同时配置和验证两者。
