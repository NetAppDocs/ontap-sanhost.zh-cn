在使用 FC 的系统上，选择 MPIO 时，需要为 Emulex 和 QLogic FC HBA 设置以下超时值。

对于 Emulex 光纤通道 HBA ：

[cols="2*"]
|===
| 属性类型 | 属性值 


| LinkTimeOut | 1. 


| 节点超时 | 10 
|===
对于 QLogic 光纤通道 HBA ：

[cols="2*"]
|===
| 属性类型 | 属性值 


| LinkDownTimeOut | 1. 


| PortDownRetransCount | 10 
|===

NOTE: Windows主机实用程序会设置这些值。有关详细的建议设置、请参见link:https://docs.netapp.com/us-en/ontap-sanhost/hu_wuhu_71_rn.html["Windows Host Utilities文档"]、并选择适用于您的Windows Host Utilities版本的安装过程。
