= 了解如何使用 FCP 和 iSCSI 配置 ONTAP SAN 主机
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以将 ONTAP 作为目标，为 FCP 和 iSCSI 配置某些 SAN 主机操作系统 (OS)。如果您的主机操作系统受 SAN Host Utilities 软件支持，则首先安装相关的 OS 主机实用程序包，其中包括 SAN 工具包。然后，对于所有主机操作系统类型，您可以为 ONTAP LUN 配置多路径设置。
